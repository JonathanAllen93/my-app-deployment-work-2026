## Create or Log in to GitHub

1. Go to https://github.com
2. Click "Sign up" if you don’t have an account.
3. If you already have one, click "Sign in"

---

##  Create a New Repository

1. Click the "plus icon" in the top right corner.
2. Select  New repository**.

---

## Enter Repository Information


1. choose epository name aka  Name of your project  
   Example: `sparta global`

2. description (optional)** – Short explanation of the project basically explains what the probject is 

3. **Visibility**

   Public – Anyone can see it  or  Private– Only you and collaborators can see it

---

### Initialize the Repository

You can select these options:

1. Add a "README file"
2. Add a "gitignore"
3. Choose a "license" 

These help organise your project.

---

 Create the Repository

Click "Create repository"

Your repository will now be created.

---

##Adding  Files

You can now add files by:

1. Clicking "Add file"
2. Selecting "Upload files"
3. Dragging your project files into the repository



If you already have a project on your computer, open the terminal and run:

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/USERNAME/REPOSITORY.git
git push -u origin main
```

Replace "USERNAME" and "REPOSITORY" with your GitHub details.
